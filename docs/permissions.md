# Permissions

All permission nodes use the `hyperessentials` root prefix.

## Homes

| Permission | Description |
|------------|-------------|
| `hyperessentials.home` | Use /home |
| `hyperessentials.home.set` | Set homes |
| `hyperessentials.home.delete` | Delete homes |
| `hyperessentials.home.list` | List homes |
| `hyperessentials.home.gui` | Open homes GUI |
| `hyperessentials.home.share` | Share homes |
| `hyperessentials.home.unlimited` | Unlimited homes |
